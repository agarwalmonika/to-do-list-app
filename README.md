# 📝 To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.
This project helps users manage daily tasks efficiently with basic CRUD operations.

🚀 Live Demo
👉 

📌 Features
➕ Add new tasks
📝 Edit existing tasks
❌ Delete tasks
📋 Dynamic task rendering
⚡ Instant updates without page refresh
🧠 Beginner-friendly vanilla JavaScript logic


🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling
JavaScript (Vanilla) – Functionality
📂 Project Structure
to-do-list-app/
│
├── index.html      # Main HTML file
├── style.css       # Styling file
└── script.js       # JavaScript logic

⚙️ How It Works
➤ Add Task
User enters a task → clicks "Add" → task is displayed in the list dynamically.

➤ Edit Task
Click edit button → task loads into input → update and save.

➤ Delete Task
Click delete button → task is removed instantly from UI.

🐛 Known Issue Fixed
Problem:

Page was refreshing when submitting a task.

Solution:

Used:event.preventDefault();
to stop default form behavior.
